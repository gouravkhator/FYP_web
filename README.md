# FYP

## Usage

### Installation Guide
```bash
pip install pipenv
```
Then clone the repo. Inside the cloned directory, run the following:
```bash
pipenv install
```

Then, add all the follwing input files in the path 'app/inputs/'.
* movies_metadata.csv
* links_small.csv

Then run from cloned directory :
```bash
python app/algos/simple_recom.py
```
